Features

Everything in one board

Designed and built from scratch — no dev kit, no shields.

🎙

Stereo MEMS mics

Dual ICS-43434 with I²S output, power-gated via load switch. 30 dB digital gain applied in firmware before streaming.

🔊

Class-D amplifiers

MAX98357A × 2 for stereo output. Direct I²S — no DAC, no I²C config. 3 W per channel into 4 Ω.

🧠

Claude AI

Streaming Anthropic API. LLM token generation and TTS synthesis run concurrently — no silence between sentences.

🔋

Battery + charging

Single-cell LiPo with USB-C charging via MCP73831T and accurate state-of-charge via MAX17048 fuel gauge.

📐

6-axis IMU

LSM6DSOXTR with hardware interrupt lines and an on-chip ML core. Ready for gesture-based wake or control.

📊

Live health dashboard

HTTP server served directly from the ESP32 on port 80. Battery, IMU, load switches, WiFi RSSI — updates every 3 s.

🖥

Browser conversation UI

Streaming token display, pipeline state badge, full transcript — all in a single vanilla JS file, no frameworks.

⚙️

FreeRTOS multitasking

mic_task pinned to core 1, speaker_task to core 0, WebSocket driven from loop(). No thread contention by design.

💡

Status LEDs

Three hardware LEDs show I²S data flow, VAD gate activity, and WebSocket connection state without opening a serial monitor.

Quick start

Get it running

You need Python 3.9+, PlatformIO, and an Anthropic API key.

1

Download TTS model files

kokoro-onnx models aren't in the repo (~170 MB). Download kokoro-v1.0.onnx and voices-v1.0.bin into server/ from the kokoro-onnx GitHub releases page.

2

Install server dependencies

Create a venv and install: python -m venv server/.venv then pip install -r server/requirements.txt. On Linux/Mac, make install does this.

3

Configure your API key

Copy .env.example → .env and set ANTHROPIC_API_KEY.

4

Flash the firmware

Copy wifi_config.h.example → wifi_config.h, add your WiFi credentials and server LAN IP, then pio run -e voicebox -t upload.

5

Start the server and speak

Run server/.venv/Scripts/python server/main.py. Open http://localhost:8080. When the badge shows listening — say something.
